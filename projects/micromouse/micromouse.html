Micromouse Maze Solver

A robot that solves mazes autonomously.

As part of the UCT-designed Micromouse to autonomously solve an 8x8 maze.
The robot hardware is run by an STM32 microcontroller, two geared DC motor-driven wheels, and uses a set of infrared sensors and an Inertial Measurement Unit (IMU).

The project scope entailed coding the robot to utilize the raw data from the IMU and IR sensors to allow the robot to accurately and reliably know where it is at all times.
The robot is then programmed to use a flood-fill algorithm to navigate the maze and find the optimal path to the finish with the least amount of exploring.

Key Features

  • Infrared wall sensor optimization for navigation
  • Maze-solving algorithm (Flood Fill)
  • STM32 programming through MATLAB Simulink and Simscape
